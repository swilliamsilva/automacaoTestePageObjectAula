# automacaoTestePageObjectAula

PAGE OBJECT
===========

Teste de Front End.

Teste da camada que o usuário vê, tudo que você consegue iterajir em um site.

Voce esta em um site e esse site pode ter muitas páginas.
Ele é um padrão que diz que o seus testes podem ser organizados por páginas.
Então você tem a possibilidade de organizar cada teste que você for fazer por página.
Isso deixa o teste mais simples e a manutenção mais fácil.
